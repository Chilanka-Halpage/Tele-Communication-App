1. Application runs on http://localhost:8080/

2. Spring boot and Angular has been used

3. Angular built file is in the jar file.

4. Lombok has been used in Spring Boot to generate getters, setters, constructors etc.

5. One to many relationships exist between tables

6. This telecommunication app helps in keeping track of customer details, their payment and package activated details.

7. Application.properties file in spring boot app contains all database configurations and accordingly they can be altered

8. Angular source code does not contain the node_modules directory. Therefore, the npm install command can be used to install necessary packages.
